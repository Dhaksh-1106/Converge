# Converge
A secure, role-based collaboration platform where students can propose, discover, and join cross-department research projects vetted by faculty administrators.

## AI Assistant Integration
The dashboard now includes a built-in AI Research Assistant powered by OpenAI. It can answer project idea questions, collaboration advice, and platform usage help.

### Setup
1. Copy `backend/.env.example` to `backend/.env`.
2. Add your OpenAI API key to `OPENAI_API_KEY`.
3. Optionally set `OPENAI_MODEL` (defaults to `gpt-3.5-turbo`).
4. Use the dashboard as a logged-in user and ask the assistant from the sidebar.
